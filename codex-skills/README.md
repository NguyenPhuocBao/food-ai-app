# Codex Skills cho VS Code

Bộ file này dùng để hướng dẫn AI Codex code frontend/backend rõ ràng hơn, tránh code chung chung, tránh lỗi kiến trúc, và giữ giao diện đẹp, đồng bộ màu sắc.

## Cách dùng nhanh

1. Copy thư mục `codex-skills` vào project.
2. Khi chat với Codex, dán nội dung file phù hợp:
   - `frontend-skill.md`: khi làm giao diện.
   - `backend-skill.md`: khi làm API/backend.
   - `ui-design-system.md`: khi muốn giao diện đẹp, màu sắc đồng bộ.
   - `fullstack-rules.md`: khi làm cả frontend + backend.
   - `codex-prompt-template.md`: mẫu prompt dùng mỗi lần yêu cầu Codex code.

## Câu lệnh mẫu

> Hãy đọc và tuân thủ file `codex-skills/frontend-skill.md` và `codex-skills/ui-design-system.md`. Sau đó code trang Login theo yêu cầu bên dưới...
