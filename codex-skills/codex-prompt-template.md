# Mẫu prompt dùng cho Codex trong VS Code

Dán mẫu này vào Codex mỗi khi muốn code chức năng mới.

```txt
Hãy đọc và tuân thủ các file sau:
- codex-skills/frontend-skill.md
- codex-skills/backend-skill.md
- codex-skills/ui-design-system.md
- codex-skills/fullstack-rules.md

Yêu cầu:
[Viết chức năng cần làm ở đây]

Công nghệ đang dùng:
- Frontend: [React/Vite/Next.js/...]
- Backend: [Node.js/Express/NestJS/Laravel/...]
- Database: [MongoDB/MySQL/PostgreSQL/...]
- Auth: [JWT/session/không có]

Quy tắc:
- Không code giao diện chung chung.
- Giao diện phải đẹp, có màu sắc đồng bộ.
- Không phá cấu trúc project hiện tại.
- Không đổi tên file/function đang được dùng nếu không cần.
- Nếu thiếu thông tin, hãy tự kiểm tra code hiện tại trước khi sửa.
- Sau khi làm xong, liệt kê file đã sửa và cách test.
```
